# SCADA Dashboard Project

## 📌 Overview
This project is a real-time SCADA dashboard for industrial automation.

## ⚙️ Features
- PLC Data Monitoring
- Real-time Dashboard (Streamlit)
- OPC UA Communication
- Data Logging

## 🛠️ Technologies Used
- Python
- Streamlit
- OPC UA
- Pandas
