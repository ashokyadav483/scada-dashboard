# SCADA Dashboard (Streamlit)

## 📌 Overview
This project is a real-time SCADA dashboard for monitoring PLC data.

## ⚙️ Features
- Live data visualization
- Real-time dashboard
- KPI display
- Trend chart

## 🛠️ Technologies
- Python
- Streamlit
- Pandas

## 🚀 Run Project
pip install -r requirements.txt  
streamlit run app.py
